Como crear un Lorem Ipsum

Probar si su editor de texto no tiene el comando lorem, si no lo tiene siga las siguientes instrucciones
  1. Ingrese a la página https://es.lipsum.com/ o a https://getlorem.com/es/
  2. Seleccione que formato quiere
    • Párrafos:
      Este formato te generará la cantidad de párrafos que pidas del texto Lorem Ipsum, sirve para grandes cantidades de textos como artículos o descripciones
    • Palabras:
      Este formato te generará la cantidad de palabras que pidas del texto Lorem Ipsum, sirve para reducidas cantidades de textos como titulares
    • Bytes:
      Este formato te generará la cantidad de letras que pidas del texto Lorem Ipsum, sirve para pequeñas cantidades de texto como títulos o etiqetas
    • Listas:
      Este formate te generará la cantidad de líneas qie pidas del texto Lorem Ipsum, sirve para listas de texto, generalmente para listas no ordenadas
  3. Luego de seleccionar el formato colocamos la cantidad de texto en ese formato que precisamos
  4. Ahora ya tenemos el Lorem Ipsum que precisamos, solo hay que copiarlo y pegarlo en donde lo precises en tu página
